# 模拟器模拟器 🎮

一个有趣的农场经营模拟游戏,扮演一名模拟器农民,在90天内售出100,000个模拟器!

## 游戏目标

- **初始库存**: 100,000个模拟器
- **时间限制**: 90天
- **胜利条件**: 售完所有模拟器
- **失败条件**: 90天后仍有库存

## 如何运行

直接在浏览器中打开 `index.html` 文件即可开始游戏!

```bash
# Windows
start index.html

# Mac
open index.html

# Linux
xdg-open index.html
```

无需安装任何依赖,纯HTML+CSS+JavaScript实现!

## 游戏特色

### 🎯 核心玩法

- **动态定价系统**: 调整价格平衡销量和利润
- **市场营销**: 投放广告提升知名度和销量
- **客户互动**: 与不同类型客户谈判交易
- **连锁店经营**: 开设多家店铺扩大销售
- **赛事活动**: 参加模拟器大赛赢得奖励
- **随机事件**: 应对天气变化和突发事件

### 📊 游戏系统

#### 定价策略
- 低价(¥2-4): 销量+80%,利润低
- 正常(¥5-7): 标准销量
- 高价(¥8-10): 销量-40%,利润高
- 溢价(¥11-15): 销量-70%,需要高声誉支撑

#### 营销渠道
- 传单派发: ¥200, 销量+15%, 持续3天
- 社交媒体: ¥500, 销量+30%, 声誉+5, 持续5天
- 本地电台: ¥1,000, 销量+50%, 声誉+10, 持续7天
- 电视广告: ¥3,000, 销量+100%, 声誉+20, 持续10天

#### 店铺类型
- 路边摊: 解锁条件5,000个, 成本¥1,000
- 农贸市场: 解锁条件15,000个, 成本¥3,000
- 社区小店: 解锁条件30,000个, 成本¥8,000
- 连锁超市: 解锁条件60,000个, 成本¥15,000
- 线上商城: 解锁条件80,000个, 成本¥10,000

#### 赛事活动
- 品质大赛: 第15、45、75天
- 创意设计大赛: 第30、60天
- 销售冠军赛: 第90天(最终评比)

### 🎮 游戏技巧

1. **初期策略**: 保持适中价格(¥5-7),积累资金
2. **营销投入**: 适时投放广告,提升声誉和销量
3. **店铺扩张**: 达到解锁条件后立即开店
4. **赛事参与**: 积极参加比赛,赢取丰厚奖励
5. **客户互动**: 与特殊客户谈判可获得额外收益
6. **天气应对**: 雨天和暴雨天销量下降,提前做好准备

## 开发者说明

游戏采用模块化设计,主要包含以下系统:

1. **状态管理**: GameState对象集中管理所有游戏数据
2. **计算引擎**: 销量、价格、声誉等核心算法
3. **UI更新**: 统一的界面刷新机制
4. **事件系统**: 客户、赛事、随机事件处理
5. **存档系统**: LocalStorage自动保存进度
